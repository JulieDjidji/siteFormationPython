Installation en local

Pour installer python en local sur son ordinateur, nous vous conseillons d’installer la distribution anaconda disponible sur le site d’Anaconda.

  • Sous Windows, il suffit de télécharger l’exécutable puis de l’exécuter. Le site vous détaille par exemple la procédure à suivre.
  • Sous Linux, le tutoriel permet par exemple d’installer anaconda.

Mais qu’est ce qu’anaconda ?

Anaconda est une distribution Python. A son installation, Anaconda installe par défaut :

  • Python
  • Une multitude de packages (la liste des packages installés par défaut avec anaconda est disponible sur cette page.

Anaconda propose aussi un outil de gestion de packages appelé Conda. Conda permet de mettre à jour et d’installer facilement les librairies nécessaires pour nos développements.
